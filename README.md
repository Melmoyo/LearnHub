# LearnHub — Learning Management System
LearnHub is a frontend learning management system (LMS) that lets users browse
courses, track their learning progress, and manage educational content in a clean,
structured interface.
## Live Demo
[View Live](https://melmoyo.github.io/Learnhub) | [View
Code](https://github.com/Melmoyo/Learnhub)
## Features
- Browse available courses by category
- Course detail view with curriculum breakdown
- Progress tracking per course
- Clean dashboard showing enrolled courses and completion status
- Responsive layout for all screen sizes
## Why I Built This
LMS platforms like Udemy and Coursera are complex products. Building a simplified
version helped me understand how to architect a multi-view application with real
navigation, state management across pages, and a content-heavy UI — all without a
framework.
## Tech Stack
- HTML, CSS, JavaScript (vanilla)
- Mock course data (JSON)
